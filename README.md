# Хотел Беденски Бани – Уебсайт

Статичен уебсайт за Хотел Беденски Бани, построен с Jekyll и съвместим с GitHub Pages.

## Как да публикувате сайта в GitHub Pages

1. Създайте ново GitHub хранилище (repository)
2. Качете (push) всички файлове в хранилището
3. Отидете в **Settings** → **Pages**
4. В секция **Source** изберете **Branch: main** (или master) и папка **/ (root)**
5. Натиснете **Save**
6. След 1–2 минути сайтът ще бъде достъпен на адрес: `https://<username>.github.io/<repo-name>/`

## Как да редактирате съдържанието

Цялото съдържание на сайта се намира в един файл: **`_data/content.yml`**

За да промените текст:
1. Отворете файла `_data/content.yml`
2. Намерете текста, който искате да промените
3. Променете текста **в кавичките** (не променяйте имената преди двоеточието)
4. Запазете файла и го качете (commit + push) в GitHub

### Примери

**Промяна на телефонен номер:**
```yaml
hotel:
  phone: "+359888999999"
  phone_display: "0888 999 999"
```

**Добавяне на нова услуга:**
```yaml
services:
  items:
    - "Минерален басейн"
    - "Рехабилитация"
    - "Настаняване"
    - "Хранене"
    - "Масаж"
```

**Добавяне на нова цена:**
```yaml
pricing:
  items:
    - name: "Стандартна стая"
      price: "50 лв / нощ"
    - name: "Пакет с лечение"
      price: "70 лв / нощ"
    - name: "Апартамент"
      price: "90 лв / нощ"
```

## Структура на проекта

```
_config.yml          - Настройки на Jekyll
_data/content.yml    - Цялото съдържание на сайта
_layouts/default.html - HTML шаблон
_includes/header.html - Хедър с навигация
_includes/footer.html - Футър
index.html           - Главна страница
assets/style.css     - Стилове (CSS)
```
